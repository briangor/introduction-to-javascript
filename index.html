In JavaScript, an object is a standalone entity, with properties and type.

Example: A student object with name and score properties

        

        

        

            
Inheritance

            

                Inheritance enables you to define a class that takes all the functionality from a parent class and
                allows you to add more
            

            
Example: A child class (Student) that inherits the parent (Person)
            

            

                
                
                
            


        

        

        

            
Encapsulation

            

                Encapsulation is the bundling of data and the methods that act on that data such that access to that
                data is restricted from outside
            

            
Example: Employee class with name and rating properties. Getter and
                Setter methods to manipulate data

            

                
                
                

                
                
                    

                
            

            

        

        

        

            
Prototypes

            

                JavaScript prototype property also allows you to add new methods to object constructors
            

        

        

        

            
Closure

            

                A closure is a function that references variables in the outer scope from its inner scope.
